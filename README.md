Currently, I work at **Amazon** as a Software Engineer, contributing to enterprise-scale products using modern technologies. At **Novo Nordisk**, I took on a senior role delivering impactful full-stack solutions in a hybrid environment. At **Illinois DCFS**, I worked on customer-facing applications leveraging Microsoft Azure and JavaScript.

My technical toolkit includes **Java, Python, Golang, JavaScript, ReactJS, Spring**, and various backend and frontend frameworks. I also have a strong interest in AI/ML and have worked on projects that blend data-driven decision-making with intelligent automation. I thrive in collaborative environments, value clean and maintainable code, and am always eager to learn new tools and technologies.
